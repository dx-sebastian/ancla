# Support

## Reporting issues

If you find a bug or want to request a feature, please open an issue on GitHub:

**https://github.com/dx-sebastian/ancla/issues**

## Useful information to include

- VS Code version
- OS and version (e.g. Windows 11)
- Steps to reproduce the problem
- Screenshot or error message if available
