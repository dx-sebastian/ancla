# Changelog

All notable changes to **Ancla – Port Manager** are documented here.

---

## [1.0.2] – 2026-05-08

### Added
- `keywords`, `bugs`, `homepage`, and `galleryBanner` fields in `package.json` for better Marketplace visibility.
- `CHANGELOG.md` and `SUPPORT.md` files.

---

## [1.0.1] – 2026-04-01

### Fixed
- Minor stability improvements.

---

## [1.0.0] – 2026-03-01

### Added
- Initial release.
- Live port list with process name and PID.
- Framework detection (React, Vue, Angular, ASP.NET, Docker, Storybook, and more).
- Smart filters: All ports, Dev ports only, or by framework.
- Kill process with a single click.
- Protection layer for critical system processes.
- Sidebar panel accessible from the Activity Bar.
- Keyboard shortcut `Ctrl+Alt+P` / `Cmd+Alt+P`.
